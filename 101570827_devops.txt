COMP2156 - DevOps Course Overview

DevOps is a methodology that combines development and operations 
to improve software delivery speed and reliability.

In COMP2156, we learn about:
- Git and GitHub
- Branching and merging
- Pull requests
- Collaboration workflows
- Continuous Integration (CI)
- Automation practices

This course helps students understand version control, 
team collaboration, and DevOps culture.

It prepares students to work in modern IT environments 
where automation and continuous delivery are important.
